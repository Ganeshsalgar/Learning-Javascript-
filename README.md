# Data Types 
# Primitive Data type : 
    A Primitive data type is an immutable, single-value data type whose value is directly represented into variable.

    JavaScript has seven primitive types: String, Number, BigInt, Boolean, Undefined, Null, and Symbol.

## characteristic :
    1. Stores a single value.
    2. Immuatable (cannot create copy)
    3. compared by Value.
    4. Assignment create copy of value
    5. Method work through temporarily object wrappers.  



# Data Types 
# Non - Primitive Data type : 
   A non-primitive data type is an object-based, mutable data type that can store collections of data and behavior. Variables hold references to these objects rather than the objects themselves. Examples include Objects, Arrays, Functions, Maps, Sets, and Dates.


## characteristic :
    1. Mutable.
    2. Stored as objects.
    3. Compared by reference.
    4. Variables store references.
    5. Can contain properties and methods.
